◆ correct_angle()

float RobotDriver::correct_angle ( )

Correct the orientation of the robot.

Makes requests to the get_correction server and calls the RobotDriver::turn method with the received angle. Continue until the server sends an angle bellow the correction threshold.

Returns
Final angle after correction.

◆ drive()

Distance RobotDriver::drive ( float  distance)

Makes the robot move in a linear way.

Sets a maximum speed and determine total time to travel the given distance. Sends Twist messages with a linear speed following a trapezoidal function to ensure smooth movement.

Note
For more details, see inline comments in robot_driver.cpp.
Parameters

Returns

bool RobotDriver::turn ( bool  clockwise,
float  radians 
)

Rotates the robot with a given angle in radians.

Based on the tutorial for the ROS package pr2_controllers. Records the transform at the start, then sends Twist messages with an angular speed following a v-shaped function to ensure smooth movement.

Note
For more details, see inline comments in robot_driver.cpp.
